workers.default.xml.xml_validator module¶

workers.default.xml.xml_validator.validate_xml(xml_file_path, dtd_file_path=None, schema_file_path=None)[source]¶

Validate XML for well-formed, XSD and DTD.

Parses the given XML file. This throws syntax error if not well formed. When the additional schema parameter is given it also checks the XML file against that. When parameter is given also checks against the DTD referenced in the XML. The DTD is downloaded from the web.

Parameters:
  • xml_file_path (str) – path to XML file to be validated
  • dtd_file_path (str) – (optional) path to DTD to be checked against
  • schema_file_path (str) – (optional) path to XSD to be checked against
Raises:
  • etree.XMLSyntaxError – if XML document is not well-formed
  • etree.DocumentInvalid – if XML document does not adhere to XSD
Returns:

None
