workers.nlp.annotate.annotate.annotate(text, lang=None)[source]¶

Annotate the given text.

Uses the TextAnalyzer of the dainst/nlp_components in the nlp-worker docker container to annotate the text.

Parameters:
  • text (str) – the given text to annotate
  • lang (str) – language of the text, if empty the text_analyzer tries to figure out it itself
Return dict:

generated annotations with metadata
